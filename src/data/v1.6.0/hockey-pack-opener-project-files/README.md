# Hockey Pack Opener - player data files

Plaats deze bestanden in je project:

- `src/data/playersBase.ts` - verrijkte basisdata met nationaliteiten
- `src/data/playersBase.json` - dezelfde data als JSON
- `src/data/playerRatings.ts` - game ratings, rarity en traits
- `src/data/nationalities-review.csv` - spelers/nationaliteiten om handmatig te controleren
- `src/data/nationality-summary.json` - samenvatting
- `scripts/validatePlayers.ts` - validatiescript

Advies:
1. Commit eerst je bestaande project.
2. Kopieer deze bestanden naar je repo.
3. Pas imports aan van `src/players.ts` naar `src/data/playersBase.ts` + `src/data/playerRatings.ts`.
4. Draai daarna het validatiescript.

Let op: nationaliteiten met status `assumed_nl` of `source_based` zijn nog niet even sterk als `verified_fih` of `verified_club`.

## Trait confidence update v1 - 2026-05-02

Alle oude `traits` arrays in `playerRatings.ts` zijn leeg gehouden. Nieuwe trait-toekenning staat in `traitScores` met per trait een confidence percentage van 0 t/m 100.

Belangrijk:
- Topscorerdata geeft hogere confidence op `schieten`, `sleepcorner`, `tip-in`, `1 tegen 1 specialist`, `dribbelaar`, `snelheid` en `rebound`.
- Keepers krijgen `keeper reflexen` op basis van KNHB/club keeper-status.
- Den Bosch Heren 1 heeft verrijkte posities op basis van HC Den Bosch-profielen.
- Spelers zonder positie of duidelijke mediahit krijgen lage default scores, zodat je later regels kunt toepassen zoals `confidence >= 60`.

Nieuwe bestanden:
- `src/data/playerRatings.ts`
- `src/data/playerRatings.withTraitConfidence.json`
- `src/data/trait-confidence-review.csv`
- `src/data/trait-confidence-summary.json`

## National team evidence update

Added on 2026-05-02: national-team and Jong Oranje evidence from Hockey.nl official team pages and news publications.

New fields in `src/data/playersBase.ts` / `.json`:

- `internationalStatuses`: source-backed evidence rows per player.
- `isSeniorInternational`: true when matched on current Hockey.nl Oranje Heren/Dames pages.
- `isJongOranje`: true when matched on the 2026 Jong Oranje training groups.
- `internationalStatusSummary`: compact app-friendly status (`senior_international`, `jong_oranje`, `historical_youth_international`, or `none_found_hockeynl`).

New files:

- `nationalTeamEvidence.ts/json/csv`
- `nationalTeamSourceRegistry.ts/json`
- `national-team-unmatched-review.csv`
- `national-team-summary.json`
