# Dataset versie-map v1.5.1

Deze map bevat de datasetbestanden voor versie v1.5.1 van de hockey pack opener dataset.

Gebruik deze map als primaire bron voor Claude/Cursor:

```txt
src/data/v1.5.1/playersBase.ts
src/data/v1.5.1/playerRatings.ts
src/data/v1.5.1/traitCatalog.ts
src/data/v1.5.1/playerCountries.ts
```

De bestanden direct onder `src/data/` blijven aanwezig voor backwards compatibility, maar nieuwe implementaties moeten naar deze versie-map verwijzen.
